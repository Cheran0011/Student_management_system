🎓 Student Management System:
-----------------------------
  A full-stack Student Management System developed using Spring Boot, Java, SQL, and Web Technologies. This application helps manage student records efficiently with features like adding, updating, deleting, and viewing student information through REST APIs.

📌 Project Description:
------------------------
  The Student Management System is a backend-based web application that allows administrators to manage student data in a structured way. The system provides RESTful APIs to perform CRUD operations and stores data in a SQL database. The APIs were tested using Postman.
  This project demonstrates backend development, database integration, and API development using Spring Boot.

🚀 Features:
------------
-> Add new student details
-> View all student records
-> Update student information
-> Delete student records
-> REST API implementation
-> Database integration
-> Tested APIs using Postman

🛠️ Technologies Used:

-> Spring Boot
